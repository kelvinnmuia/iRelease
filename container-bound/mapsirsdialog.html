Map Releases SIRs

Map SIRs from Bugzilla to specific releases and iterations.

How it works

Enter SIR IDs from release notes. The system will:

  • Look up each SIR in the SIRs sheet
  • Generate unique SRL-XXXXX IDs for new mappings
  • Skip duplicates (same SIR + Release + Iteration)
  • Update only changed fields for existing mappings

Enter the release version (e.g., 24.3.0, 25.1.0)

Enter the iteration identifier

Enter comma-separated SIR IDs. Each ID will get a unique SRL-XXXXX identifier.

Example Input:

Release Version: 24.3.0

Iteration: I1

SIR IDs: 128135, 128179, 128234

Will create: SRL-7AGUX for 128135, SRL-CV3FN for 128179, etc.
